Java基础

面向对象的创造,使用,封装

前言

该分栏主要是发布一些Java的基础，非常适用于各新手学习和复习。

因为我在这几天时间里自学到了Java面向对象的内容，所有直接更新我现学习的阶段，前面的内容我会以后慢慢补上。

本篇主要介绍Java面向对象的创造，使用，封装等。

概念

Java 是面向对象的编程语言，对象就是面向对象程序设计的核心。所谓对象就是真实世界中的实体，对象与实体是一一对应的，也就是说现实世界中每一个实体都是一个对象，它是一种具体的概念。
