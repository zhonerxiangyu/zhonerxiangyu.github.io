Java基础

Java || 参数的传递

前言

该分栏主要是发布一些Java的基础，非常适用于各新手学习和复习。

本篇主要介绍Java面向对象参数的传递思路和逻辑。

概念

参数传递主要研究和学习的是方法在调用的时候，涉及到参数传递的问题。

到底是怎么传递数据的呢？

答： 值得传递

int i = 10;

int j = i; //i传递给就，实际上质数将i变量中保存的10（值）传递给了j，j实际上是一块全新的内存空间。

User u = 0x123456;

User u2 = u; //u传递给u2，实际上是将0x123456这个值赋值给了u2，u和u2实际上是两个不同的局部变量，
但是它们这两个变量指向堆内存中同一个Java对象。

结论

方法调用的时候，涉及到参数传递的问题，传递的时候，Java只遵循一种语法机制就是将变量中保存的”值”传递过去了，只不过有的时候这个值是一个字面值10，有的时候这个值是另一个Java对象的内存地址0x123456。
